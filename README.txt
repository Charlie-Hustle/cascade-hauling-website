Cascade Hauling Co. Website

Files included:
- index.html
- styles.css
- script.js
- assets/cascade-logo.png

How to use:
1. Upload all files and folders to your GitHub repository.
2. Keep index.html in the main/root folder.
3. Keep styles.css and script.js in the same folder as index.html.
4. Keep the assets folder with cascade-logo.png inside it.

To update contact info:
Search index.html for charles@cascadehaulingco.com and replace it if needed.

Note:
The contact form currently opens the visitor's email app. For a real web form later, connect it through Formspree, Netlify Forms, or Google Forms.
